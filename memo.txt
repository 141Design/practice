ローカルリポジトリ作成
    cd 新しくローカルリポジトリを作るディレクトリ
    git init

ローカルリポジトリにコミットする
    git add 追加するファイル
    git commit -m "add : memo.txt"

リモートリポジトリにプッシュする
    初回：
    git remote add origin https://github.com/141Design/practice.git
    以降：
    git push origin develop

ブランチを作る
    git branch <branchname>

ブランチを切り替える(新しいブランチを作って移動する)
    git checkout <branch>
    

マージする(マージ元に移動する)
    git merge branch
